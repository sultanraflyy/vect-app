# Vect Blog

Static blog for [tryvect.com/blog](https://tryvect.com/blog).

## Deployment

This blog is deployed as a static site on Vercel.

### Setup

1. Create a new Vercel project
2. Import the `sultanraflyy/vect-app` repository
3. Set **Root Directory** to `web/blog`
4. Set **Framework Preset** to "Other" (static site)
5. Add custom domain: configure as needed for `tryvect.com/blog`

### Auto-Deploy

Vercel automatically deploys when changes are pushed to `web/blog/`.
The GitHub Actions workflow `blog-deploy.yml` validates blog posts on push.

## Adding Blog Posts

1. Create a new `.md` file in `web/blog/posts/`
2. Include frontmatter:
   ```yaml
   ---
   title: "Post Title"
   slug: "url-friendly-slug"
   date: "YYYY-MM-DD"
   description: "Meta description for SEO (150-160 chars)"
   keywords: ["keyword1", "keyword2"]
   author: "Vect"
   ---
   ```
3. Write the article content in Markdown
4. Commit and push — Vercel auto-deploys

## Content Generation

Blog posts are also auto-generated by the Content Engine:
- Run: `node agents/engine.js --channel blog`
- Posts are saved to both `web/blog/posts/` and `content/blog/`
- The GitHub Actions `content-generation.yml` workflow handles this weekly
