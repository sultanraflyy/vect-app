How it works

Four steps from raw content to verified report.

  1. 1

    Paste, upload, or drop a URL

    Submit any text blob, PDF document, or web URL. Vect ingests it instantly.

  2. 2

    Claims extracted & grouped

    Vect's AI identifies every verifiable claim and intelligently deduplicates near-identical assertions.

  3. 3

    AI agents research each claim

    Parallel AI agents hit live sources — academic databases, news archives, official records — and rate confidence for each claim.

  4. 4

    Trust Score + full audit report

    Receive a 0–100 Trust Score, per-claim verdicts, confidence ratings, and sourced evidence. Export as PDF or CSV.

Built for developers

Integrate claim verification directly into your product with the Vect REST API.

Request

POST https://api.vect.app/api/verify 
{
  "text": "The moon landing happened in 1969.",
  "max_claims": 10,
  "deep_research": false
}

Response · 200 OK

200 application/json 
{
  "trust_score": 94,
  "status": "verified",
  "claims": [
    {
      "text": "Moon landing in 1969",
      "verdict": "TRUE",
      "confidence": 0.99,
      "sources": ["nasa.gov"]
    }
  ]
}
