# Content Archive

This directory stores all generated content organized by channel.
Content is auto-generated by the Vect Content Engine and committed via GitHub Actions.

## Structure

```
content/
├── reddit/        # r/SaaS discussion posts (manual posting only)
├── linkedin/      # Thought leadership posts (distributed via Buffer)
├── x-threads/     # 7-tweet contrarian threads (distributed via Typefully)
├── instagram/     # Visual narrative posts (distributed via Buffer)
├── outreach/      # Cold email + LinkedIn DM templates (manual send)
├── community/     # Expert witness Slack/Discord responses (manual post)
├── blog/          # SEO micro-blog articles (auto-deployed via Vercel)
└── signals/       # Trend detection data and generation summaries
```

## Content Lifecycle

1. **Generated** — Content Engine creates channel-specific content
2. **Archived** — Content is committed to this directory
3. **Validated** — Voice model validation is run
4. **Distributed** — Content is sent to platforms via automation or manual posting
5. **Tracked** — Engagement signals feed back into trend detection

## File Naming Convention

```
YYYY-MM-DD-theme-name.md
```

Example: `2026-03-13-the-sovereignty-gap.md`

## Important Notes

- **Reddit content is NEVER auto-posted** — always manually reviewed and posted
- **Community responses are NEVER auto-posted** — always manually shared
- **Outreach templates require manual personalization** before sending
- Blog content auto-deploys to tryvect.com/blog via Vercel on git push
